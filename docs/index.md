# Computational Systems & Generative Worlds

Welcome! 👋  
This site documents our journey through **creative coding, generative systems, and interactive digital environments** using **Processing / py5 (Python Mode)**.

This is both:
- 📘 A **learning resource**
- 🎨 A **portfolio of example projects**
- 🧠 A map of how ideas build on each other

---

## 🧭 How to Use This Site

### If You Are a Student:
- Start at **Fundamentals**
- Move forward **in order**
- Each folder represents a **new concept or skill**
- Every project includes:
  - ✅ The code
  - ✅ A short explanation
  - ✅ Ideas for extending it

You are **not expected to understand everything at once**.

---

### If You Are Reviewing This Work:
This site shows progression from:
- simple drawing
- to interaction
- to systems
- to installation‑scale tools

## 🌿 Project Concept

The projects focus on **living systems** — code that adapts to its environment.

These projects combine:
- Sound
- Vision
- Motion
- Space
- Behavior

They are designed to work in **real architectural contexts**, not just on screen.
